Secure HTTP (HTTPS)

Last updated

This data measures whether GOV.UK support the HTTPS protocol (https://), and the strength of that support. HTTPS provides a secure connection across the internet between websites and their visitors, and is becoming the new baseline for public web services.

Note that HTTPS generally does not affect whether a website is vulnerable to hacking.
